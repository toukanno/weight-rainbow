プライバシーポリシー

アプリ名: Rainbow Weight Log（レインボー体重管理）

最終更新日: 2026年3月28日

1. 収集する情報

本アプリは、ユーザーが入力した以下の情報を端末内にのみ保存します。

  • 名前（表示用ニックネーム）
  • 身長・体重・年齢・性別
  • 体重記録の履歴

これらのデータは端末のローカルストレージに保存され、外部サーバーへの送信は行いません。

2. カメラ・マイク・写真ライブラリの使用

本アプリは体重入力の補助として、カメラ撮影、写真ライブラリからの画像読み込み、マイクによる音声入力、および音声認識機能を使用します。これらの機能で取得したデータ（画像・音声）は端末内の入力処理にのみ使用され、外部へ送信されることはありません。

3. 第三者への情報提供

本アプリは、ユーザーの個人情報を第三者に提供、販売、または共有しません。

4. 広告・トラッキング

現在のビルドには広告SDK、トラッキングコード、またはATT（App Tracking Transparency）の仕組みは含まれていません。将来的に広告を導入する場合は、本ポリシーを更新し、適切な同意取得を行います。

5. データの削除

アプリを端末から削除することで、保存されたすべてのデータが削除されます。

6. お問い合わせ

本ポリシーに関するお問い合わせは、以下のGitHubリポジトリのIssuesページよりお願いします。

https://github.com/toukanno/weight-rainbow/issues

Privacy Policy

App: Rainbow Weight Log

Last updated: March 28, 2026

1. Information We Collect

This app stores the following user-entered information locally on the device only:

  • Name (display nickname)
  • Height, weight, age, and gender
  • Weight record history

All data is stored in the device's local storage and is never transmitted to external servers.

2. Camera, Microphone, and Photo Library

This app uses the camera, photo library, microphone, and speech recognition as input assistance for weight entry. Data captured by these features (images and audio) is used only for on-device input processing and is never sent externally.

3. Third-Party Sharing

This app does not provide, sell, or share any personal information with third parties.

4. Advertising and Tracking

The current build does not include any advertising SDK, tracking code, or App Tracking Transparency (ATT) mechanism. If advertising is introduced in a future update, this policy will be updated and appropriate consent will be obtained.

5. Data Deletion

Deleting the app from the device removes all stored data.

6. Contact

For questions about this policy, please use the Issues page of the GitHub repository below.

https://github.com/toukanno/weight-rainbow/issues
